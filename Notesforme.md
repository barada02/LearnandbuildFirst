## Step 1: Initializing your Rust Project
First, we need to bootstrap a new Rust binary (executable) project in our workspace directory.

```bash
cargo init --bin
```

#### What does this do?
* `cargo` is Rust's official package manager, build tool, and dependency coordinator.
* `init` initializes a new Cargo package in an existing directory.
* `--bin` tells Cargo that this will be a **binary application** (an executable program with a `main` function) rather than a library.
